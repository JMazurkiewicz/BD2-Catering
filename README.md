# BD2-Catering

## Aplikacja

### Instalowanie zależności

W systemie Windows:

```batch
pip install -r tools/requirements.txt
```

W systemie Linux:

```bash
pip install -r tools/requirements.txt
sudo apt-get install python3-tk
```

### Uruchamianie aplikacji

```bash
cd app/src
python main.py
```

### Uruchamianie testów

```bash
pytest app/tests
```
